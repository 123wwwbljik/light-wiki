# 光年之外

> 一个基于 Docsify 的伪星际档案馆。设定全靠瞎编，但排版尽量像那么回事。

## 宇宙底稿

`光年之外` 记录的是一个不太严肃、但内部自洽的宇宙。

这里的文明以档案形式保存，每个角色、地点和事件都像是从某本遗失的星图边角里抄出来的注释。页面风格不再走默认文档模板，而是按“档案馆”方向整理成一个更像世界观入口的首页。

> “在浩瀚宇宙里，最先被证伪的往往不是谎言，而是常识。”

## 核心住民

### 档案号 001：似锦

| 字段 | 内容 |
| --- | --- |
| 姓名 | 似锦 |
| 种族 | 赛博佩奇猪 |
| 籍贯 | 仙女座星系 · 浪漫粉尘星云 · 哼哼星 |
| 状态 | 活跃，且非常饿 |
| 危险等级 | 中等偏香 |

似锦并非一只平凡的猪。它在出生时与一块上古微型智能芯片发生了不可解释的量子缠绕，于是顺手获得了超光速干饭、跨维度拱地和低频加密破解等能力。

它最著名的一句发言是：

> “宇宙的尽头不是黑暗，是地瓜。”

### 档案号 002：折光邮差

一位长期穿梭在废弃航线中的匿名信使。没人知道他真实长相，因为每次抵达空间站时，他的脸都会像被劣质棱镜处理过一样发生偏折。

已知特征：

- 从不迟到，但经常提前三天送达昨天的信件。
- 使用过期星图导航，准确率反而高得离谱。
- 与似锦存在未公开的食堂合作关系。

## 异能目录

### 赛博哼哼

低频声波可直接干扰简陋加密协议，也能震落树上的成熟果实。前者用于情报战，后者用于午餐。

### 量子拱地

通过对地面进行高强度拱击，制造短暂且不太稳定的微型虫洞。一般用于赶路，也偶尔用于逃单。

### 似锦之光

在吸收恒星能量后，毛发表面会产生绸缎般的反光层。没有实战意义，但极具档案封面价值。

## 航线与区域

### 浪漫粉尘星云

以常年漂浮的高饱和粉尘闻名。视觉效果很好，但飞船过滤器报废率也很高。

### 无尽地瓜星系

传说中由可食用小行星带组成的偏远区域。许多探险日志提到过它，但没有一份能在返航后保持逻辑完整。

### 第七码头

位于边缘星港的灰色中转区，大部分地下情报交易和奇怪餐车都在这里发生。
